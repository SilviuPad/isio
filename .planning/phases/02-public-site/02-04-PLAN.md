Card layout -- Fiverr-style tier card:
- If `isRecommended`: outer border `border-2 border-electric-500 relative` + absolute badge at top center: `<span class="absolute -top-3 left-1/2 -translate-x-1/2 bg-electric-500 text-white text-xs font-bold px-4 py-1 rounded-full">Recomandat/Recommended</span>` (per PRIC-06).
- If NOT recommended: `border border-navy-200`.
- Shared styles: `rounded-2xl p-8 bg-white`.
- Header: Name in `text-xl font-bold font-heading text-navy-950`. Price in `text-4xl font-bold text-navy-950` with "EUR" suffix in `text-lg text-navy-400`. If pricePrefix exists, show it above price in `text-sm text-navy-500`.
- Description in `text-navy-500 text-sm mt-2`.
- Delivery: "{deliveryDays} zile livrare" / "{deliveryDays} day delivery" in `text-sm text-navy-400`.
- Revisions: if -1 show "Revizuiri nelimitate"/"Unlimited revisions", else "{revisions} revizuiri"/"{revisions} revisions". Use `text-sm text-navy-400`.
- Features list (`mt-6 space-y-3`): Each feature has a checkmark (green `text-emerald-500`) or X mark (gray `text-navy-300`) based on `included` boolean. Text in `text-sm text-navy-700` if included, `text-navy-400 line-through` if not.
- CTA button at bottom: If recommended `bg-electric-500 hover:bg-electric-600 text-white`, else `bg-navy-100 hover:bg-navy-200 text-navy-700`. Full width, `py-3 rounded-full font-semibold text-center`. Text from i18n `pricing_select`.
